# Definition of Done

## 1. 要件充足
- PBIに記載されたAcceptance Criteriaをすべて満たしている
- 不明点が残っていない

## 2. 実装
- コードがローカルで正常に動作する
- エラーが発生しない

## 3. テスト
- 最低限の正常系テストが存在する
- 主要処理が確認できている

## 4. 可読性
- コードが過度に複雑でない
- 関数が単一責任を意識している

## 5. ドキュメント
- 必要に応じてREADMEやコメントが更新されている

## 6. 状態更新
- PBIのStatusがDoneになっている
- product-backlog.mdが更新されている
- sprintファイルが更新されている
